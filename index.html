Proyek dan Studi Kasus

Sistem Kartu Stok Barang

Aplikasi Laravel untuk mencatat pergerakan stok barang berdasarkan gudang, produk, dan ukuran. Termasuk perhitungan otomatis kolom sumqty dan sumrupiah menggunakan trigger SQL dan job scheduler.

Tech: Laravel, MySQL, Eloquent

Fitur: otomatisasi trigger, perhitungan dinamis, relasi kompleks antar tabel

Dynamic View Generator

Sistem untuk menampilkan view dinamis berdasarkan data dari controller tanpa membuat banyak file blade manual. View menyesuaikan data yang dikirim secara otomatis.

Tech: Laravel Blade, Controller Dynamic Data

Fitur: fleksibilitas tampilan, efisiensi maintenance

Trigger Journal Otomatis

Membuat class PHP dinamis untuk menghasilkan trigger SQL otomatis yang menjurnal transaksi ke tabel kartuakun, termasuk logika PPN, pengecekan status jurnal, dan perhitungan ulang nilai.

Tech: Laravel Migration, SQL Trigger, Dynamic Class

Fitur: pembuatan trigger otomatis melalui migration

AJAX Error Handler

Implementasi jQuery AJAX di Laravel untuk menampilkan error SQL dari Exception di browser menggunakan alert, tanpa reload halaman.

Tech: Laravel, jQuery AJAX, JSON Exception

Fitur: notifikasi error real-time di sisi client

Data Relasi & Grouping Otomatis

Eksperimen relasi kompleks menggunakan `hasManyThrough`, `groupBy`, dan model dinamis untuk menghindari data duplikat dan meningkatkan efisiensi query.

Tech: Laravel Eloquent, Relational Query

Fitur: pengelompokan dinamis dan relasi lintas model
